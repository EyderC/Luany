# Luany
Personal AI agent For Algorithmic Trading An Continuous Learning.

## Overview
Luany is a dual-purpose AI agent: a specialized algorithmic trading system for prop firm challenges (BTC/USD, EUR/USD, NQ, ES) and personal tutor for python and technical English in the context of AI engineering.

## Tech Stack 
Python | FastAPI | ChromaDB | Docker | Github Actions

## Markets
BTC/USD | EUR/USD | E-mini Nasdaq 100 (NQ) | E-mini S&P 500 (ES)
