# API

FastAPI service that exposes Luany's core functionality
as documented HTTP endpoints.

## Endpoints (planned)
- `POST /risk-check` — Validates a trade against challenge rules
- `GET /challenge-status` — Returns current drawdown and P&L
- `POST /chat` — Send a message to Luany agent

## Docs
Run the server and visit `/docs` for the auto-generated 
Swagger UI.
